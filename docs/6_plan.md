Критерии завершенности каждого этапа:
Модель данных: Проходит все юнит-тесты сериализации
FileManager: Корректно читает/пишет данные разного размера
Index система: Обеспечивает O(1) поиск, обрабатывает коллизии
DatabaseEngine: Выполняет все CRUD операции из ТЗ
HTTP API: Все эндпоинты возвращают корректные ответы
Фронтенд: Все функции доступны через UI
Дополнительные функции: Backup/export работают стабильно
